App Entry Points

Primary Entry Point

  • SplashActivity → Checks authentication → Routes to LoginActivity or DashboardActivity

Deep Links

  • PrivacyPolicyActivity - Deep link from web (/privacy-policy)
  • ExitAppActivity - Exit handler (no display)

Navigation Patterns

Pattern 1: Bottom Navigation

  • Persistent across: Dashboard, Messages, Scanner, Child Care, Students
  • 5 tabs: HOME, MESSAGE, SCANNER, CHILDCARE, STUDENT
  • Behavior: Always visible, switches between main modules

Pattern 2: Side Drawer Navigation

  • Available in: All BaseNavigationActivity screens
  • Structure: Expandable list with groups and children
  • Access: Hamburger menu icon in toolbar

Pattern 3: Tab Navigation

  • Used in:
    • ScannerActivity (Staff/Student/Cab tabs)
    • Messages (Inbox/Sent/Scheduled tabs)
    • Create Student (Multi-step tabs)

Pattern 4: List → Detail Navigation

  • Pattern: List screen → Click item → Detail screen
  • Examples:
    • StudentActivity → StudentDetailActivity
    • AlbumActivity → AlbumDetailActivity
    • InboxMessageActivity → InboxMessageDisplayActivity

Pattern 5: Create/Edit Flow

  • Pattern: List → Create/Edit → Save → Back to List
  • Examples:
    • StaffActivity → CreateStaffActivity → StaffActivity
    • ClassActivity → CreateClassActivity → ClassActivity

Pattern 6: Multi-Step Forms

  • Pattern: Wizard-style navigation with progress indicator
  • Examples:
    • CreateStudentNewActivity (5 steps)
    • CreateMessageActivity (Compose → Recipients → Send)

Pattern 7: Modal/Overlay Screens

  • Pattern: Full-screen overlays for specific actions
  • Examples:
    • ImageZoomActivity
    • ImageAnnotationActivity
    • AudioMessageRecord

Pattern 8: Deep Linking

  • Pattern: Direct navigation from external sources
  • Examples:
    • PrivacyPolicyActivity (from web)
    • Notification deep links

Migration Notes for React Native

Navigation Structure

  • Use React Navigation with:
    • Bottom Tab Navigator (5 tabs)
    • Drawer Navigator (side menu)
    • Stack Navigator (list → detail flows)
    • Tab Navigator (within screens like Scanner)

Screen Mapping

  • Each Activity → React Native Screen Component
  • Fragments → Can be components or separate screens
  • Dialogs → Modal components

State Management

  • User authentication state
  • Navigation state
  • Feature flags (rights, modules)
  • Current user context
